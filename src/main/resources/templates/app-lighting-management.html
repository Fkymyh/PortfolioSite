Java / Swing

照明機材管理アプリ

照明機材の一覧管理や検索、現場ごとの使用機材の登録を 分かりやすくするために制作したデスクトップアプリです。

画面イメージ

機材情報を一覧で確認し、必要な機材を検索・管理できる画面を 作成しています。

制作した理由

イベント現場では多数の照明機材を扱うため、機材情報や使用状況を 分かりやすく管理する必要があります。

前職での経験をもとに、機材の検索や現場単位での登録を行える 管理アプリを作りたいと考えました。

使用技術

  • Java
  • Java Swing
  • オブジェクト指向
  • ファイル入出力
  • Git / GitHub

主な機能

  • 照明機材の一覧表示
  • 機材情報の登録・編集
  • 機材名や分類による検索
  • 現場ごとの使用機材登録
  • 機材情報の保存・読み込み
