Python / Flask

映画検索Webアプリ

映画情報を検索し、気になる作品を管理するために制作した Webアプリです。

画面イメージ

キーワードから映画を検索し、作品情報を確認できるようにしています。

制作した理由

自分が映画を見ることが好きだったため、興味のある題材を使って WebアプリとAPI連携を学びたいと考えました。

使用技術

  • Python
  • Flask
  • HTML / CSS
  • 外部API
  • Git / GitHub

主な機能

  • キーワードによる映画検索
  • 映画情報の一覧表示
  • 映画の詳細情報表示
  • お気に入り作品の管理

起動方法と公開状況

現在はWeb上への公開は行っていません。 実行環境を用意し、リポジトリの手順に従って起動すると利用できます。
