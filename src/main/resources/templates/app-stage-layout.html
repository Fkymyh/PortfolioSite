Stage Layout Designer

学生団体やイベント現場向けに制作している、ステージ仕込み図作成アプリです。 機材を直感的に配置し、会場レイアウトを分かりやすく作成することを目的にしています。

画面イメージ

左側の機材パレットから機材を選び、中央のキャンバスに配置してレイアウトを作成します。 グリッド表示により、位置関係を確認しながら配置できます。

作った理由

大学時代の放送部や前職のイベント現場で、仕込み図や機材配置の共有に手間を感じた経験がありました。 専門的なソフトを使わなくても、学生団体や後輩が直感的に使える簡易的なレイアウト作成ツールを作りたいと考えました。

主な機能

  • 音響・照明・舞台機材の配置
  • ドラッグによる移動
  • グリッド表示と吸着
  • 右クリックによる削除
  • 機材情報の編集
  • 保存・読み込み機能

使用技術

  • Java
  • Swing
  • オブジェクト指向
  • ファイル入出力
  • Git / GitHub

工夫した点

ステージ設営に詳しくない人でも使いやすいように、機材を選んで配置するだけでレイアウトを作れる構成にしました。 また、グリッドやルーラーを表示することで、位置関係を視覚的に把握しやすくしています。

苦労した点

機材のドラッグ移動や選択状態の管理、保存・読み込み時のデータ構造の整理に苦労しました。 特に、画面上の操作と内部データを一致させる部分は、実際に動かしながら何度も調整しました。

また、CanvasPanelに描画処理やマウス操作の処理が集まり、クラスが大きくなってきたため、 今後は処理を分割して、保守しやすい構成に改善したいと考えています。

今後追加したい機能

  • ズーム機能
  • テンプレートの追加
  • CanvasPanelの分割による保守性改善
