学習記録

職業訓練や個人制作を通して学んだ内容をまとめています。 単に学んだ技術名だけでなく、制作物のどこで活かしたかも整理しています。

Java

基本文法、オブジェクト指向、コレクション、ファイル入出力などを学習しました。 照明機材管理アプリやStage Layout Designerの制作で活用しています。

Java Swing

デスクトップアプリの画面作成、ボタン操作、マウス操作、描画処理などを学びました。 Stage Layout Designerでは、機材の配置、ドラッグ移動、グリッド表示などに使用しています。

Servlet / JSP

リクエスト処理、画面遷移、フォーム処理、スコープ、MVCの基本を学びました。 Webアプリケーションの基本的な流れを理解するための土台になっています。

Spring Framework / Spring Boot

現在、このポートフォリオサイトの制作を通して学習しています。 ControllerでURLを受け取り、Thymeleafで画面を表示する構成を実装しています。

Python / Flask

映画検索Webアプリの制作で使用しました。 APIを利用した映画検索や、Web画面での表示処理を学びました。

Git / GitHub

制作物の履歴管理や公開に使用しています。 変更内容をコミットし、GitHub上でソースコードを公開できるようにしています。
